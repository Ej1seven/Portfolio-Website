Project Overview

Quote Generator is an application I designed with the purpose of generating random quotes that you can post on your Twitter account. I created an asynchronous function which extracts quotes from https://type.fit/api/quotes. By selecting the "New Quote" button you can obtain a new quote from the API. By selecting the Twitter button, you will access your Twitter account where your Twitter message will automatically be fed with the quote you selected in the API. This function allows you to select the Post button from your Twitter account without having to copy and paste the quote. My main goal when creating this application was to display my ability to extract data from an API in an effective way.
